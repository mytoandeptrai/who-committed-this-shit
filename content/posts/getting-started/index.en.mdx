## Introduction

This is the first post on my personal blog. Built with **Next.js 15**, **MDX**, and **Tailwind CSS v4**.

## Tech stack

- **Next.js 15** with App Router
- **MDX** via `next-mdx-remote` + `gray-matter`
- **Tailwind CSS v4** for styling
- **shadcn/ui** for UI components
- **Shiki** for syntax highlighting

## Code example

```typescript
import { getAllPosts } from '@/modules/blog/lib/posts';

const posts = await getAllPosts('en');
console.log(posts.length); // number of posts
```

## Conclusion

Simple blog, content-first. No admin UI, no database — just push to git and it deploys.
